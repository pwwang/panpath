# S3 Paths API

API reference for Amazon S3 paths.

::: panpath.s3_sync.S3Path
    options:
      show_root_heading: true
      show_source: true

::: panpath.s3_async.AsyncS3Path
    options:
      show_root_heading: true
      show_source: true

## Overview

S3 paths provide access to Amazon S3 storage.

## Sync Usage

```python
from panpath import PanPath

path = PanPath("s3://bucket/file.txt")
content = path.read_text()
```

## Async Usage

```python
from panpath import AsyncPanPath

path = AsyncPanPath("s3://bucket/file.txt")
content = await path.read_text()
```
